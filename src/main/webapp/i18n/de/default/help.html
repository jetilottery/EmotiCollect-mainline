EmotiCollect

Platzen Sie vor Freude während Sie Emoticons sammeln, um große Preise zu gewinnen! Drehen Sie das Rad und sehen Sie zu, wie passende Emoticons platzen und die Emoticon-Anzeige füllen. Füllen Sie eine Anzeige, um zu gewinnen! Wenn das Rad auf einem Sternsymbol landet, wählen Sie eine Sternkarte für einen Sofortgewinn! Manche Emoticons decken zusätzliche Spiele für noch mehr Gewinnchancen auf!

Wie wird gespielt?

Ticketpreis

Zeigt den aktuellen Ticketpreis an.

Drücken Sie auf den linken Pfeil (–), um den Ticketpreis zu verringern.

Drücken Sie auf den rechten Pfeil (+), um den Ticketpreis zu erhöhen.

Drücken Sie auf KAUFEN, um den Ticketpreis vom Guthaben abzuziehen und das Spiel zu spielen.

Spielanleitung

Bei jedem Spiel wird das Rad 5 Mal gedreht. Drücken Sie auf DREHEN, um das Rad zu drehen und ein Emoticon-Symbol oder ein Sofortgewinn-Sternsymbol zu sammeln.

Wenn Sie ein Emoticon-Symbol sammeln, werden alle passenden Emoticons auf dem Raster platzen und einen Teil der entsprechenden Emoticon-Anzeige füllen. Füllen Sie eine Anzeige, um einen Preis zu gewinnen.

Wenn Emoticons auf dem Raster platzen, enthüllen manche davon ein Zusatzspiel-Symbol, das der Anzeige ein zusätzliches Spiel hinzufügt.

Sammeln Sie ein Sofortgewinn-Sternsymbol, um eine Sofortgewinnkarte zu wählen. Wählen Sie eine Karte aus, um den entsprechenden Preis zu gewinnen.

Das Spiel endet, wenn alle Spiele abgeschlossen wurden. Eine Meldung erscheint, die das Ergebnis des Spiels anzeigt.

Alle aufdecken

Drücken Sie auf ALLE AUFDECKEN, um das Spiel automatisch bis zum Ende zu spielen. Das Spiel geht weiter, bis alle Spiele beendet sind.

Regeln

Missbrauch oder Fehlfunktion annulliert alle Gewinne und Spiele.

Sofortgewinne werden zum Gesamtgewinnbetrag addiert.

Gewinne werden in der ausgewählten Währung angezeigt.

Spielerentscheidungen, die während des Spiels gemacht werden, beeinflussen nicht das Ergebnis des Spiels.
